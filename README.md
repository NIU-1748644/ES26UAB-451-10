# QueMenges - Team ES26UAB-451-10

## Descripción del Proyecto
El proyecto consiste en el diseño de **QueMenges**, una aplicación que permite a los usuarios acceder a menús preparados por personas individuales (cocineros), diferenciándose de plataformas de grandes superficies. La aplicación gestionará cuentas de usuario, realización y envío de pedidos, pagos y un sistema de valoraciones.
