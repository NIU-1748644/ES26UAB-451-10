# Requirement

### ID:
RF-3-05

### Type/Category:
Funcional

### Version:
1.0

### Title:
Visualización de la posición del repartidor

### Description:
El sistema deberá mostrar al cliente la posición actual del repartidor
en un mapa interactivo durante el proceso de entrega.

### Relations:


### Comments:
La posición del repartidor se actualizará mientras el envío esté en curso.
