# Requirement

### ID:
RF-4-01

### Type/Category:
Funcional

### Version:
1.0

### Title:
Selección del método de pago

### Description:
El sistema deberá permitir al usuario seleccionar el método de pago de una comanda.
Las opciones disponibles serán:
- pago mediante tarjeta bancaria,
- pago mediante la plataforma PayPal.

### Relations:
RF-4-02

### Comments:
La selección del método de pago deberá realizarse antes de confirmar la transacción.
