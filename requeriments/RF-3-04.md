# Requirement

### ID:
RF-3-04

### Type/Category:
Funcional

### Version:
1.0

### Title:
Seguimiento del envío

### Description:
Una vez que el pedido haya sido pagado y el envío haya comenzado,
el sistema deberá permitir al cliente visualizar el progreso del envío
de su pedido desde la aplicación o página web.

### Relations:


### Comments:
La funcionalidad de seguimiento estará disponible únicamente cuando
el pedido tenga el estado "Pagado".
