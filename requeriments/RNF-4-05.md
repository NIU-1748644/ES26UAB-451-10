# Requirement

### ID:
RNF-4-05

### Type/Category:
No funcional

### Version:
1.0

### Title:
Integración con pasarelas de pago

### Description:
El sistema deberá integrarse con APIs externas de entidades bancarias
y con la plataforma PayPal para procesar los pagos realizados por los usuarios.

### Relations:
RF-4-02

### Comments:
La integración deberá realizarse mediante las APIs oficiales de cada plataforma.
