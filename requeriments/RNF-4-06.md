# Requirement

### ID:
RNF-4-06

### Type/Category:
No funcional

### Version:
1.0

### Title:
Gestión segura de datos de autenticación

### Description:
Todas las credenciales y datos de autenticación de pago deberán gestionarse
mediante conexiones seguras con las plataformas externas de pago.

### Relations:


### Comments:
El sistema no deberá almacenar credenciales sensibles del usuario.
