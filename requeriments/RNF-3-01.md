# Requirement

### ID:
RNF-3-01

### Type/Category:
No funcional

### Version:
1.0

### Title:
Integración con Google Maps

### Description:
El sistema deberá integrar la visualización del mapa utilizando la API
de Google Maps para mostrar la posición del repartidor durante el
seguimiento del envío.

### Relations:
RF-3-05

### Comments:
La integración deberá realizarse mediante la API oficial de Google Maps.
